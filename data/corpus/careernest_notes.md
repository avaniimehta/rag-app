# CareerNest Project Notes

CareerNest is a full-stack job portal built with React, Node.js, Express,
and MySQL. It uses Socket.IO for real-time notifications and JWT for
authentication. The platform integrates a local LLM (Ollama running Mistral)
to help with resume-to-job-description matching and interview preparation.

Key features include employer job posting, candidate applications, real-time
chat between recruiters and candidates, and AI-generated interview questions
based on the job description.
